# Frontend - Startuate

Modern Vue 3 + TypeScript frontend with a minimalistic black and white design.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Runs on http://localhost:5173

## Build

```bash
npm run build
```

## Tests

```bash
npm run test
npm run test:ui
npm run test:coverage
```

## Design

- **Minimalistic**: Clean, focused interface
- **Black & White**: High contrast, professional aesthetic
- **Modern**: TypeScript, Vue 3 Composition API, Vite
- **Responsive**: Mobile-first design
